# Star-Blood-Unity

The game was created thanks to the Unity game engine. Game genre: 2D,RPG, Roguelike. There are 3 heroes in the game, for each of which you can play. 3 various locations with completely random generation of the world, enemies are located on biomes, when they are destroyed, the player is transferred to the next location. There are 3 difficulty levels (easy, normal, difficult) in which you can try yourself. There is a system of improvements, without which the player will not pass to the next location. Kill enemies, collect crystals and reach the end!
